# Add-on Icon

To complete the add-on, you should add an icon file:

- Create `icon.png` (128x128 pixels) with a water meter or water drop icon
- Create `logo.png` (optional, for add-on store)

You can use free icons from:
- Material Design Icons (https://materialdesignicons.com/)
- Choose a water-related icon like `mdi:water-pump` or `mdi:water-circle`
- Export as PNG at 128x128 pixels

Alternatively, you can use any image editing software to create a simple icon with:
- Blue/cyan colors for water theme
- A water drop, meter gauge, or pipe symbol
- WAZ or water meter representation
